# Contributing

Thanks for contributing to Axelliant React Enterprise Starter.

## Development Workflow
1. Fork the repo and create a feature branch.
2. Install dependencies: `npm install`.
3. Run local checks: `npm run lint`, `npm run typecheck`, `npm run test`.
4. Open a pull request with a clear summary.

## Commit Guidelines
- Use concise, descriptive commit messages.
- Keep changes focused and avoid unrelated refactors.

## Code Style
- Follow ESLint and Prettier outputs.
- Use TypeScript strict mode rules.
- Keep modules small and single-purpose.

## Testing
- Add tests for new functionality.
- Keep unit tests fast and deterministic.

## Security
- Never commit secrets or credentials.
- Report security issues to `security@axelliant.com`.
