# Security Policy

## Supported Versions
This repository is maintained on the `main` branch. Security patches are applied to the latest release only.

## Reporting a Vulnerability
Please report security issues privately to `security@axelliant.com`.

Include:
- A clear description of the issue
- Steps to reproduce
- Any potential impact

We aim to acknowledge reports within 2 business days and provide a remediation plan within 7 business days.
